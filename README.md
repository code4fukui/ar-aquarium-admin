# ar-aquarium-admin

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

An administration interface for the [ar-aquarium](https://github.com/code4fukui/ar-aquarium) project.

## Demo
Access the [ar-aquarium-admin](https://code4fukui.github.io/ar-aquarium-admin/) with a URL parameter (e.g. https://host/api/path1/path2/).

## Features
- Allows adding, editing, and deleting fish in the AR aquarium
- Provides options to customize fish behavior and appearance
- Saves fish data to a [json-storage](https://github.com/code4fukui/json-storage/) backend

## Requirements
- Set up a [json-storage](https://github.com/code4fukui/json-storage/) backend

## Usage
1. Set up the [json-storage](https://github.com/code4fukui/json-storage/) backend.
2. Access the [ar-aquarium-admin](https://code4fukui.github.io/ar-aquarium-admin/) with a URL parameter (e.g. https://host/api/path1/path2/).
