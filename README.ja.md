# ar-aquarium-admin

[ar-aquarium](https://github.com/code4fukui/ar-aquarium)のための管理画面です。この管理画面では、AR水族館に表示する水生生物を追加、編集、削除できます。また、水生生物の表示設定（種類、サイズ、動き）の変更や、[json-storage](https://github.com/code4fukui/json-storage/)への水生生物データの保存が可能です。

## 機能
- 水生生物の追加、編集、削除
- 水生生物の表示設定（種類、サイズ、動き）の変更
- [json-storage](https://github.com/code4fukui/json-storage/)への水生生物データの保存

## 必要環境
- [json-storage](https://github.com/code4fukui/json-storage/)のセットアップ

## 使い方
1. [json-storage](https://github.com/code4fukui/json-storage/)をセットアップします。
2. URLパラメーターに[json-storage](https://github.com/code4fukui/json-storage/)のパスを指定して[ar-aquarium-admin](https://code4fukui.github.io/ar-aquarium-admin/)にアクセスします（例: https://host/api/path1/path2/）。
